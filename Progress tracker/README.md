## How to Use

1. **Open the Application**: Simply open `index.html` in your web browser
2. **Set Your Targets**: Adjust the daily targets for each activity (work in hours, others in minutes)
3. **Track Progress**: As you complete activities, update the "completed" fields
4. **Add Notes**: Record what you accomplished in each area
5. **Save Progress**: Your data auto-saves, but you can manually save anytime
6. **View History**: Click "View History" to see your past performance
7. **Reset if Needed**: Use "Reset Day" to start over (be careful - this can't be undone!)
